My Résumé in LaTeX format

This is a page for my Github repository hosting the original source .tex and the accompanying font and image files.

I have also provided a PDF version, for easy download, exported from the source files.
I used the service WriteLatex for the PDF conversion.

One of the great features of the .tex file, it can be converted to almost any file type, including RTF.

Please feel free to download my résumé and distribute it as much as you like.
